# 📁 Datasets

Shared datasets used across all modules. Each dataset is stored once here and referenced from notebooks.

| File | Size | Used In | Description |
|------|------|---------|-------------|
| `breast-cancer-wisconsin.csv` | 125 KB | Classification → KNN | Wisconsin breast cancer diagnostic data (569 samples, 30 features) |
| `diabetes.csv` | 24 KB | Feature Engineering → Embedded Methods | Pima Indians Diabetes dataset (768 samples, 8 features) |
| `heart_disease_uci.csv` | 79 KB | Classification Metrics | UCI Heart Disease dataset |
| `height-weight-simple.csv` | 4 KB | Regression → Simple Linear | Simple height-weight dataset for linear regression basics |
| `housing-data.csv` | 35 KB | Feature Engineering → Wrapper Methods | Housing price prediction data |
| `human-activity-recognition.csv` | 19 MB | Feature Engineering → Filter Methods | Smartphone sensor data for HAR |
| `insurance.csv` | 56 KB | Regression → Comprehensive Linear | Medical insurance cost prediction |
| `modified-synthetic-economic-data.csv` | 43 KB | Regression → Multiple Linear | Synthetic economic indicators |
| `studentscores.csv` | 214 B | Regression basics | Simple student scores dataset |
| `playtennis.csv` | 5 KB | Decision Trees | Classic PlayTennis dataset |
| `IMDB Dataset.csv` | 66 MB | Naive Bayes → Sentiment Analysis | 50K movie reviews for NLP |
| `mnist-digit-recognizer.csv` | 77 MB | Neural Networks (future) | MNIST handwritten digits |
| `energy_weather_raw_data.csv` | 77 MB | Data Preparation (future) | Energy and weather data |

> **Note:** Large files (>50 MB) — consider using [Git LFS](https://git-lfs.github.com/) for `IMDB Dataset.csv`, `mnist-digit-recognizer.csv`, and `energy_weather_raw_data.csv`.
