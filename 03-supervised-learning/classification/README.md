# 🏷️ Classification Algorithms

Each algorithm has its own subfolder with numbered notebooks and guides.

> **💡 Dual-purpose algorithms:** KNN, Decision Trees, and SVM can also be used for regression tasks. The implementations here focus on classification. Regression variants can be found (or added) under `../regression/` in the future.

## Algorithms

| Folder | Algorithm | Key Concepts |
|--------|-----------|--------------|
| `knn/` | K-Nearest Neighbors | Distance metrics, k selection, curse of dimensionality |
| `logistic-regression/` | Logistic Regression | Sigmoid, cross-entropy loss, OvR, softmax, MLE |
| `naive-bayes/` | Naive Bayes | Bayes theorem, conditional independence, Laplace smoothing |
| `decision-tree/` | Decision Trees | Entropy, Gini impurity, pruning, visualization |
| `svm/` | Support Vector Machines | Hyperplanes, margins, kernels |
